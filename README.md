# Scientific GUI Calculator in C

A modern **Scientific Calculator with Graphical User Interface (GUI)** built using the **Win32 API in C**.

This project combines a futuristic dark-themed interface with powerful scientific and mathematical operations.

---

# ✨ Features

## 🖥️ Modern GUI Interface
- Custom Win32 GUI design
- Rounded neon-glow buttons
- Dark futuristic theme
- Separate expression & result display
- Large responsive display screen
- Owner-drawn custom buttons
- Smooth color-coded layout

---

# 🔢 Basic Operations
- Addition
- Subtraction
- Multiplication
- Division
- Decimal calculations
- Power calculations
