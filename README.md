# Scientific Calculator in C

A feature-rich scientific calculator built in C with support for multi-value computations, advanced math functions, and efficient command-line interaction.

## ✨ Features

Scientific_Calculator.c
│
├── Basic Calculations (add, sub, mul, div)
├── Trigonometry Module
├── Hyperbolic Module
├── Exponentiation & Log Module
├── Combinatorics Module
├── Helper Functions
│   ├── factorial()
│   ├── toRadians()
│   └── toDegrees()
└── Main Menu System
